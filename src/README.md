# Next Play – Performance Task Board

Next Play is a polished Kanban-style performance operations board inspired by tools like Asana, Linear, and Notion. It supports anonymous guest users, drag-and-drop task management, team assignment, comments, activity tracking, and AI-powered task enhancement.

## Live Features

- Modern sports-inspired UI
- Splash entry experience
- Supabase anonymous guest sessions
- Row Level Security (RLS)
- Drag and drop across workflow columns
- Create, edit, and delete tasks
- Team members and assignees
- Task comments
- Activity log timeline
- Search and filtering
- Due date indicators
- AI-powered task enhancement with OpenAI
- Coach insight panel and board stats

## Tech Stack

- React
- Tailwind CSS
- Vite
- Supabase
- Node.js
- Express
- OpenAI API
- dnd-kit
